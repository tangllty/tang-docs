# Vercel 网站分析介绍

Vercel 网站分析为您的 Web 应用程序提供注重隐私的分析功能。它可以帮助您了解访问者如何与您的网站互动，而不会损害他们的隐私。

## 什么是 Vercel 网站分析？

Vercel 网站分析是直接构建在 Vercel 平台中的轻量级、注重隐私的分析解决方案。它提供有关网站流量和性能的基本洞察，而不使用 Cookie 或收集个人数据。

## 主要功能

- **隐私优先**：无 Cookie，不收集个人数据，完全符合 GDPR、CCPA 和其他隐私法规
- **轻量级**：对页面加载时间影响极小，脚本大小极小
- **实时数据**：在 Vercel 仪表板上实时查看您的分析数据
- **路由跟踪**：支持的框架自动跟踪页面浏览和导航
- **自定义事件**：跟踪用户交互，如按钮点击、表单提交等（专业版和企业版计划）
- **零配置**：在 Vercel 项目上启用后即可自动工作

## 为什么选择 Vercel 网站分析？

### 注重隐私

与传统的分析工具不同，Vercel 网站分析：
- 不使用 Cookie
- 不跨网站跟踪用户
- 不收集个人身份信息 (PII)
- 默认符合隐私法规

### 框架集成

Vercel 网站分析提供与流行框架的无缝集成：
- **Next.js**（Pages 和 App Router）
- **Remix**
- **Nuxt**
- **SvelteKit**
- **Astro**
- **Vue**
- **React**
- 纯 HTML 网站

### 易于实施

快速开始很简单：
1. 在 Vercel 项目仪表板中启用网站分析
2. 安装 `@vercel/analytics` 包
3. 将 Analytics 组件添加到您的应用程序
4. 部署并开始跟踪

## 您可以跟踪什么

### 页面浏览
跟踪网站上的每次页面浏览，包括：
- 总访问者数
- 唯一访问者
- 每页的页面浏览量
- 流量来源

### 自定义事件
在专业版和企业版计划中，您可以跟踪：
- 按钮点击
- 表单提交
- 下载
- 自定义用户交互

### 性能指标
了解网站的性能：
- 页面加载时间
- 交互时间
- 核心 Web 指标
