# 简介

Kite 是一个使用 Kotlin 编写的轻量级 ORM 框架，旨在简化数据库操作的开发。它支持多种数据库，如 PostgreSQL、MySQL、Derby 等，并提供了多种 API 来方便数据库操作的开发。
