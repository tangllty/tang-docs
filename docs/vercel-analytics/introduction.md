# Introduction to Vercel Web Analytics

Vercel Web Analytics provides privacy-friendly analytics for your web applications. It helps you understand how visitors interact with your site without compromising their privacy.

## What is Vercel Web Analytics?

Vercel Web Analytics is a lightweight, privacy-focused analytics solution built directly into the Vercel platform. It provides essential insights about your website's traffic and performance without using cookies or collecting personal data.

## Key Features

- **Privacy-First**: No cookies, no personal data collection, fully compliant with GDPR, CCPA, and other privacy regulations
- **Lightweight**: Minimal impact on page load times with a tiny script size
- **Real-time Data**: View your analytics data in real-time on the Vercel dashboard
- **Route Tracking**: Automatic tracking of page views and navigation for supported frameworks
- **Custom Events**: Track user interactions like button clicks, form submissions, and more (Pro and Enterprise plans)
- **Zero Configuration**: Works automatically once enabled on your Vercel project

## Why Choose Vercel Web Analytics?

### Privacy-Focused

Unlike traditional analytics tools, Vercel Web Analytics:
- Doesn't use cookies
- Doesn't track users across sites
- Doesn't collect personally identifiable information (PII)
- Complies with privacy regulations by default

### Framework Integration

Vercel Web Analytics offers seamless integration with popular frameworks:
- **Next.js** (Pages and App Router)
- **Remix**
- **Nuxt**
- **SvelteKit**
- **Astro**
- **Vue**
- **React**
- Plain HTML sites

### Easy to Implement

Getting started is simple:
1. Enable Web Analytics in your Vercel project dashboard
2. Install the `@vercel/analytics` package
3. Add the Analytics component to your app
4. Deploy and start tracking

## What You Can Track

### Page Views
Track every page view on your site, including:
- Total visitors
- Unique visitors
- Page views per page
- Traffic sources

### Custom Events
On Pro and Enterprise plans, you can track:
- Button clicks
- Form submissions
- Downloads
- Custom user interactions

### Performance Metrics
Understand your site's performance:
- Page load times
- Time to interactive
- Core Web Vitals
