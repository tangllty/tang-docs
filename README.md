## 项目简介

tang-docs 是基于 VitePress 的糖猫猫权限管理系统文档

您可以阅读[文档](https://tangllty.eu.org/)获取更多信息

* 采用 VitePress

## 环境要求

* Node.js 18.15.0

## 项目启动

1. ```bash
   yarn && yarn run docs:dev
   ```
2. 访问 [http://localhost:5173/](http://localhost:5173/)
